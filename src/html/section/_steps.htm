Этапы разработки сайта

Предпроектный анализ

Оценка требований. Проводим опрос для понимания задач и особенностей бизнеса. Изучаем конкурентов. Определяем целевую аудиторию

Проектирование

Собирается базовое семантическое ядро. Разработка карты сайта, список разделов и страниц. Подготовка и согласование эскиза сайта.

Прототипирование

Разрабатывается каркас сайта. Определяем какие блоки будут на сайте, в какой последовательности будут располагаться, логотип, телефон, почта, социальные сети. Вносятся правки согласнов ваших пожеланий.

Разработка дизайна

На основе прототипа разрабатывается дизайн. Выбирается цветовая гамма, шрифты, композиция элементов на экранах. Отрисовываются иконки, обрабатываются изображения.

Программирование

По макету разработанному дизайнером верстается сайт. При необходимости разрабатывается структура баз данных, програмируется бизнес логика.

Тестирование

Проверка отсутствия ошибок в коде (html, css, js), работа элементов согласно задания и дизайн макета, кросбраузерность. Тестирование работы сервеной части.

Запуск проекта

Проверка и покупка доменного имени. Выбор хостинга, установка сайта на сервер.

Продвижение и развитие

Анализ позиций в поисковой выдаче. Рекомендации по редактированию (дополнению) семантики, контента, блоков, страниц сайта.
